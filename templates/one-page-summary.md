# ONE-PAGE TRADING SUMMARY

==================================================
SETUPS
==================================================

EMT – Breakout continuation from 15-minute range  
Trend Continuation – Non–15-minute level break + pullback  
15m Re-Entry – False break of 15-minute range → re-entry reversal  
REM – Failed reversal → trend resumes (double cross)  
Reversal – End-of-trend or range extreme reversal with structure  
Trendline Touch – T3 entry along clean trendline

==================================================
MARKET CYCLES → VALID SETUPS
==================================================

Trend Cycle → Trend Continuation, Trendline, EMT  
Rotational Cycle → 15m Re-Entry, REM, Reversal, Trendline  
Hybrid Cycle → EMT, 15m Re-Entry, REM, Trend Continuation  
Momentum/Breakout Cycle → EMT (primary), Trend Continuation  
Trap/REM Cycle → REM (primary), 15m Re-Entry, Reversal  
End-of-Trend Cycle → Reversal, 15m Re-Entry, REM  

==================================================
DAILY EXECUTION FLOW
==================================================

Pre-market:
- Fill cycle-check template  
- Determine today’s cycle  
- Choose one primary setup  
- List setups to avoid  

During session:
- Trade only setups valid for the identified cycle  
- If live action contradicts, switch to the matching cycle family  

End of day:
- Fill evening review  
- Update trade log  
- Update cycle notes if regime is shifting
